Apresentação sobre a Tag <nav>

A tag <nav> define áreas de navegação contendo links importantes do site.

<nav>
<a href="/html/">HTML</a> |
<a href="/css/">CSS</a> |
<a href="/js/">JavaScript</a> |
<a href="/python/">Python</a>
</nav>

Como utilizar

01- O <nav>elemento destina-se apenas a blocos principais de links de navegação.
02- Leitores de tela e outras tecnologias assistivas usam a tag <nav> como um ponto de referência para ajudar usuários com deficiência visual.
03- A <nav>tag também suporta os atributos globais.
04- A <nav>tag também suporta os atributos de evento.
